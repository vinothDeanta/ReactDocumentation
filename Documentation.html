useEffect

useEffect is used to do Hook Life cyles concept insiede the functional Component. useEffect is one of the React Hooks. React Hooks is mainly used for functional Component.

  1. useEffect is one of the React Hooks. React Hooks is mainly used for functional Component.
  2. useEffect same as ComponentDidMount
  3. useEffect can be call multilple times
  4. useEffect contain accept two arguments. first argument is function and second argument is dependency array
  5. dependency array value is empty means it will call only once
  6. if dependency array contain any state value means, it will call the function every time state can change the value.

useRef
